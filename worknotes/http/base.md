# 网络协议基础

## 经典五层模型

- 应用层（HTTP/FTP...）
- 传输层（TCP/IP）
- 网络层
- 数据链路层
- 物理层

## 发展过程

### HTTP 0.9

- 只有 GET 请求
- 没有 header
- 服务器发送完就关闭 TCP 连接

### HTTP 1.0

- 增加了 POST 等命令
- 增加了 status code 和 header
- 多字符集、多部分发送、权限、缓存等等

### HTTP 1.1

- 持久链接（keep-alive）
- 增加了 pipeline 管线化
- 增加了 host 及其他命令

### HTTP 2.0

- 所有数据以二进制传输
- 同一个连接里发送多个请求不再按顺序来
- 头信息压缩及推送等提高效率的功能
